# CS Knowledge Graph 项目日志

## 项目概述
计算机科学知识图谱系统，初期以C++知识为核心，逐步扩展到其他计算机科学领域。

## 技术栈
- 开发语言: Python 3.9 (conda环境: cs_kg_env)
- 图数据库: Neo4j
- 后端框架: Flask/FastAPI
- 前端: React + D3.js
- 主要依赖: py2neo, pandas, networkx, spacy

## 项目结构
```
CS_Knowledge_Graph/
├── src/
│   ├── core/           # 核心业务逻辑
│   ├── data/           # 数据处理和导入
│   ├── api/            # API接口
│   ├── visualization/  # 可视化组件
│   └── models/         # 数据模型
├── data/               # 原始数据和配置
├── tests/              # 测试代码
├── docs/               # 文档
├── frontend/           # 前端代码
└── config/             # 配置文件
```

## 核心功能模块
1. 知识管理模块 - 增删改查、批量导入、数据验证
2. 查询引擎模块 - 图查询、路径搜索、推荐算法
3. 可视化模块 - 交互式图谱、知识网络图、学习路径
4. API接口模块 - RESTful API、查询接口、数据接口
